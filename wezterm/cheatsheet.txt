
  tmux Keybindings  (prefix: Ctrl+B)

  Sessions
    d          detach
    s          session picker
    $          rename session

  Windows (Tabs)
    c          new window
    ,          rename window
    1-9        jump to window
    n / p      next / prev window
    &          close window

  Panes (Splits)
    |          split horizontal
    -          split vertical
    h j k l    navigate panes
    z          zoom pane
    x          close pane
    Space      cycle layouts
    arrows     resize (with Ctrl held)

  Scrollback
    [          enter copy mode (q to exit)
    / ?        search fwd / back (in copy mode)

  Shell Aliases
    ta <name>  attach session
    tl         list sessions
    tn <name>  new session
    tk <name>  kill session

  WezTerm  (leader: Ctrl+Space)
    ?          this cheat sheet
    !          animation

  Press any key to close.
